# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-08

### Added
- **Debouncing utilities**: `Debouncer`, `AsyncDebouncer`, and `debounce` function
- **Throttling utilities**: `Throttler`, `AsyncThrottler`, and `throttle` function
- **Rate limiting utilities**: `RateLimiter`, `AsyncRateLimiter`, and `rateLimit` function
- **Queuing utilities**: `Queuer`, `AsyncQueuer`, and `queue` function
- **Batching utilities**: `Batcher`, `AsyncBatcher`, and `batch` function
- **Retrying utilities**: `AsyncRetryer` and `asyncRetry` function
- **Comprehensive example app** with interactive demos for all utilities
- **Full API documentation** with usage examples
- **Type-safe implementations** with proper error handling
- **ChangeNotifier integration** for reactive state management
- **Configurable options** for all utilities (wait times, limits, strategies, etc.)

### Features
- **TanStack-inspired API**: Familiar patterns from TanStack Query
- **Async support**: All utilities support both synchronous and asynchronous operations
- **State management**: Built-in state tracking and monitoring
- **Flexible configuration**: Customizable options for different use cases
- **Performance optimized**: Efficient implementations with minimal overhead
- **Well-tested**: Comprehensive test coverage and validation

### Documentation
- **Complete README** with installation and usage instructions
- **API reference** with detailed parameter descriptions
- **Interactive examples** demonstrating real-world usage patterns
- **Migration guide** for users familiar with similar libraries
